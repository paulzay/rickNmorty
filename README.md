## Rick N Morty

> This is a webapp to fetch a list of all the locations in the show, view/edit character details and search for locations, filter location by episode and character

[Live](https://rick-nmorty-eta.vercel.app/)

## Project

## Built with

- Nextjs14
- Graphql
- Apollo Client
- Tailwind CSS
- Typescript
- LocalStorage
